# Руководство по админ-панели АТИИ

## Доступ к админке

Откройте `/admin` в браузере или нажмите кнопку "Admin" в навигации.

---

## Разделы админки

### 1. **Dashboard**
- Общая статистика проектов, шаблонов, страниц
- Быстрый доступ к последним изменениям
- Обзор активности

### 2. **Редактор схем (Schemas)**
- Визуальный редактор workflow-схем для шаблонов
- Drag & Drop узлов (trigger, process, api, notification, complete)
- Сохранение схем для каждого шаблона
- Автоматическая синхронизация с шаблонами

### 3. **Управление контентом (Content)**
- **Редактирование страниц:**
  - **Главная (Home)**: Hero, Features, Projects, Solutions, Capabilities, Stats
  - **О нас (About)**: Hero, Stats, Values, Technologies
  - **Под ключ (Custom)**: Hero, Workflow Steps, Advantages, Case Studies
  
- **Редакторы для каждой секции:**
  - Hero: Badge, Title, Subtitle, Description, CTA
  - Features: Заголовок, описание, ссылка, иконка
  - Projects: Название, категория, изображение, описание, технологии
  - Solutions: Заголовок, описание, ссылка, иконка
  - Capabilities: Заголовок, описание, изображение, иконка
  - Stats: Значение, описание, иконка
  - Values: Заголовок, описание, иконка, цвет градиента
  - Technologies: Список технологий
  - Workflow Steps: Название, описание, длительность, детали, иконка
  - Advantages: Заголовок, описание, иконка
  - Case Studies: Название, описание, технологии

- **Все изменения сохраняются в localStorage и синхронизируются в реальном времени**

### 4. **Шаблоны (Templates)**
- Управление готовыми решениями
- CRUD операции: создание, редактирование, удаление, дублирование
- Фильтрация по статусу (активные/неактивные)
- Поиск по названию и описанию
- Настройка workflow для каждого шаблона

### 5. **Портфолио (Portfolio)**
- Управление проектами в портфолио
- CRUD операции: создание, редактирование, удаление, дублирование
- Фильтрация по категориям (E-commerce, Dashboard, EdTech, Management, Marketplace, Corporate)
- Фильтр "Избранные"
- Поиск по названию и описанию
- Настройка:
  - Название проекта
  - Клиент
  - Описание
  - URL
  - Изображение
  - Категория
  - Технологии
  - Избранное (featured)

### 6. **Настройки (Settings)**
- Название сайта
- Описание
- Email
- Телефон
- Адрес
- Социальные сети
- Мета-теги (SEO)
- Аналитика (Google Analytics, Яндекс.Метрика)

## Функции

### Поиск
- Глобальный поиск в правом верхнем углу
- Работает для портфолио и шаблонов
- Поиск по названию и описанию

### Экспорт/Импорт данных
- **Экспорт**: Сохраняет все данные (портфолио, шаблоны, страницы, настройки, схемы) в JSON-файл
- **Импорт**: Загружает данные из JSON-файла
- Формат файла: `atii-backup-YYYY-MM-DD.json`
- **Используйте для резервного копирования или переноса данных между окружениями**

### Синхронизация в реальном времени
- Все изменения в админке сразу отображаются на сайте
- Работает между вкладками браузера
- Использует localStorage + custom events

### Валидация
- Проверка обязательных полей при сохранении
- Предупреждения при удалении данных
- Подтверждение при импорте (замена текущих данных)

## Workflow работы

### Добавление нового проекта в портфолио:
1. Перейдите в раздел **Portfolio**
2. Нажмите **"Добавить проект"**
3. Заполните форму:
   - Название
   - Клиент
   - Описание
   - URL
   - Изображение (URL)
   - Категория
   - Технологии (через запятую)
   - Избранное (checkbox)
4. Нажмите **"Создать проект"**
5. Проект сразу появится на странице `/portfolio`

### Редактирование контента страницы:
1. Перейдите в раздел **Content**
2. Выберите страницу (Главная, О нас, Под ключ)
3. Нажмите **"Редактировать"**
4. Откроется редактор со всеми секциями страницы
5. Измените нужные поля
6. Нажмите **"Сохранить изменения"**
7. Изменения сразу отобразятся на сайте

### Создание нового шаблона:
1. Перейдите в раздел **Templates**
2. Нажмите **"Добавить шаблон"**
3. Заполните форму
4. Перейдите в **Schemas** для настройки workflow
5. Выберите шаблон и создайте схему
6. Шаблон появится на странице `/templates`

## Технические детали

### Хранение данных
- **localStorage** (ключи: `atii_websites`, `atii_templates`, `atii_pages`, `atii_settings`, `atii_workflow_schemas`)
- Автоматическое сохранение при изменении
- Синхронизация через `storage` events и custom events

### Структура данных

#### Website (Portfolio)
```typescript
{
  id: string;
  name: string;
  client: string;
  description: string;
  url: string;
  image: string;
  category: string;
  technologies: string[];
  featured: boolean;
}
```

#### Template
```typescript
{
  id: string;
  title: string;
  description: string;
  customizable: string[];
  workflow: WorkflowStep[];
  status: 'active' | 'inactive';
}
```

#### PageContent
```typescript
{
  id: string;
  name: string;
  sections: number;
  updated: string;
  content: {
    hero?: { badge, title, subtitle, description, ctaText, ctaLink };
    features?: Array<{ id, title, description, link, icon }>;
    projects?: Array<{ id, title, category, image, description, tech }>;
    solutions?: Array<{ id, title, description, link, icon }>;
    capabilities?: Array<{ id, title, description, image, icon }>;
    stats?: Array<{ id, value, label, icon }>;
    values?: Array<{ id, title, description, icon, color }>;
    technologies?: string[];
    workflowSteps?: Array<{ id, title, description, duration, details, icon }>;
    advantages?: Array<{ id, title, description, icon }>;
    caseStudies?: Array<{ id, title, description, tech }>;
  };
}
```

## Советы

1. **Регулярно делайте экспорт данных** для резервного копирования
2. **Используйте поиск** для быстрого нахождения проектов/шаблонов
3. **Фильтры** помогают организовать большое количество данных
4. **Дублирование** ускоряет создание похожих проектов/шаблонов
5. **Синхронизация** работает автоматически — изменения сразу видны на сайте

## Горячие клавиши (будущее)

- `Ctrl + S` — сохранить изменения
- `Ctrl + E` — экспорт данных
- `Ctrl + /` — фокус на поиск
- `Esc` — закрыть модальное окно
